# Nothing SDM OTA Images Archive  

🚀 A comprehensive collection of unmodified stock OTA images for [Nothing](https://nothing.tech/) & upcoming [CMF by Nothing](https://cmf.tech) devices powered by the Snapdragon Mobile (SDM) platform. For Mediatek-based models, please refer to the [nothing_mtk_archive](https://github.com/spike0en/nothing_mtk_archive) instead.
